** tính năng **
- Đăng nhập / Đăng ký
    + Đăng ký tài khoản
    + Đăng nhập tài khoản
- Quản lý tài khoản
    + Thay đổi thông tin cá nhân
    + Đổi mật khẩu
- Quản lý tên miền
    + Tìm kiếm tên miền
    + Đăng ký tên miền
    + Gia hạn tên miền
- Quản lý giỏ hàng
    + Thêm tên miền vào giỏ hàng
    + Xóa tên miền khỏi giỏ hàng
    + Cập nhật số lượng tên miền trong giỏ hàng
    + Xem giỏ hàng
- Quản lý hóa đơn
    + Xem hóa đơn
    + Thanh toán hóa đơn
    + Xuất hóa đơn
    + Huy bỏ hóa đơn
- Quản lý giao dịch
    + Thanh toán hóa đơn
    + Xem giao dịch
